# Contributing to HERA

Thank you for your interest in contributing to HERA!

## Reporting Bugs
Please open an issue on GitHub if you encounter any bugs. Include:
- A clear description of the issue.
- Steps to reproduce.
- Your MATLAB version and OS.

## Pull Requests
1. Fork the repository.
2. Create a new branch for your feature or fix.
3. **Run Tests**: Before submitting, ensure all tests pass by running:
   ```matlab
   run('tests/run_tests.m')
   ```
4. Submit your Pull Request.

## Code Style
- Follow standard MATLAB coding conventions.
- Document your code with comments.
